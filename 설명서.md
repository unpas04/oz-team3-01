## 기획 배경

### 왜 만들었나

요즘 불경기 속에서 오히려 IP 굿즈, 인형뽑기, 캐릭터 사업에 대한 관심이 폭발적으로 늘고 있다.
사람들이 힘들수록 좋아하는 캐릭터에 더 몰입하고, 팬덤 정체성을 드러내고 싶어하는 심리가 강해지기 때문이다.

이 프로젝트는 그 심리를 활용해 **"나는 진짜 팬인가?"** 라는 질문을 던지는 퀴즈 웹서비스다.
수익화보다 **바이럴**을 먼저 목표로 잡았다.

### 바이럴 포인트

- 결과에 **IP별 커스텀 등급 타이틀**을 붙여 공유 욕구 자극
  → "나 포켓몬 마스터 받았어" "귀멸 최종선발 탈락 ㅋㅋ" 같은 텍스트가 SNS에서 자연스럽게 퍼짐
- 결과를 **이미지 카드로 저장** → 인스타 스토리 공유에 최적화
- **재응시 불가 + 노말/하드 2단계** → "나 하드도 도전해봐야겠다" 재방문 유도
- 팬덤이 명확한 IP(산리오·포켓몬·귀멸 등) 타겟 → 팬들이 자발적으로 공유

### 타겟

- 10~20대, 일본 애니/캐릭터 팬덤
- SNS 활동이 활발하고 덕질 정체성을 공유하는 데 거부감 없는 유저

### 기술 스택 선택 이유

- **HTML / CSS / JavaScript 만 사용**
  → 4명이 처음 협업하는 부트캠프 프로젝트이므로 기술 스택을 최소화해 분업에 집중
  → 별도 서버 없이 정적 파일만으로 배포 가능 (GitHub Pages 등)

## 페이지 간 데이터 전달 (URL 파라미터)

```
index.html
  → quiz.html?category=pokemon&mode=hard
      → result.html?category=pokemon&mode=hard&score=16
          → share.html?category=pokemon&mode=hard&score=16&grade=고인물
```

| 파라미터 | 값 예시 | 설명 |
|----------|---------|------|
| category | sanrio / pokemon / aot / kimetsu / fma | 선택한 IP |
| mode | normal / hard | 난이도 |
| score | 0 ~ 20 | 이번 퀴즈 획득 점수 |
| grade | 전설의 덕후 / 고인물 / ... | 등급 텍스트 |

---

## localStorage 키 규칙

```
dk_{category}_{mode} = 점수 (숫자)

예)
dk_pokemon_normal = 8
dk_pokemon_hard   = 14
dk_sanrio_normal  = 10
```

- 퀴즈 완료 시 저장, **재응시 불가** (값이 있으면 버튼 비활성화)
- 종합 덕력 게이지는 저장된 모든 값을 합산해서 계산
- 만점: 150점 (카테고리 5개 × 노말10점 + 하드20점)

## 점수 체계

| 모드 | 문제 수 | 정답당 점수 | 카테고리 만점 |
|------|--------|------------|--------------|
| 노말 | 10문제 | 1점 | 10점 |
| 하드 | 10문제 | 2점 | 20점 |
| 합계 | — | — | 30점 |

- 전체 만점: **150점**
- 문제 풀: 난이도별 20문제 중 랜덤 10개 출제

## 퀴즈 데이터 형식 (data/*.js)

각자 담당 파일을 아래 형식으로 작성할 것:

```js
const QUIZ_DATA = {
  normal: [
    // 총 20문제 작성
    { q: "쿠로미의 리본 색은 검정색이다.", a: true  },
    { q: "시나모롤은 고양이 캐릭터다.",   a: false },
    // ... 20개
  ],
  hard: [
    // 총 20문제 작성 (노말보다 훨씬 어렵게)
    { q: "쿠로미의 생일은 10월 31일이다.", a: true  },
    // ... 20개
  ]
};
```

- `q`: 문제 텍스트 (YES/NO로 답할 수 있는 문장으로)
- `a`: 정답 (`true` = YES, `false` = NO)
- 노말/하드 각각 **정확히 20개** 작성

## common.css 사용법

모든 HTML 파일 `<head>`에 반드시 추가:

```html
<link rel="stylesheet" href="common.css" />
```

주요 클래스:

| 클래스 | 용도 |
|--------|------|
| `.card` | 흰색 둥근 카드 |
| `.btn.btn-normal` | 노말 버튼 (파란색) |
| `.btn.btn-hard` | 하드 버튼 (핑크색) |
| `.btn.btn-primary` | 메인 버튼 (그라데이션) |
| `.modal-overlay` | 모달 배경 |
| `.modal-card` | 모달 카드 |
| `.gauge-wrap` | 덕력 게이지 |
| `.timer-wrap` | 타이머 바 (quiz.html용) |
| `.mode-badge.normal` | 노말 뱃지 |
| `.mode-badge.hard` | 하드 뱃지 |
| `.mode-badge.done` | 완료 뱃지 (녹색) |
