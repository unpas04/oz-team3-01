## S01 — 메인 페이지 (index.html)

**역할:** 서비스 진입점. 카테고리 선택 및 덕력 현황 확인

### 구성 요소

| 영역 | 요소 | 설명 |
|------|------|------|
| 헤더 | 서비스명 | "덕력 감별소" (EF_jejudoldam 폰트) |
| 헤더 | 서브타이틀 | "나의 진짜 덕력을 확인해봐" |
| 헤더 | 태그 | "CHARACTER QUIZ" 뱃지 |
| 게이지 | 덕력 게이지 바 | 누적 점수 / 150점, 퍼센트로 표시 |
| 게이지 | 현재 등급 뱃지 | 입문자 / 라이트 팬 / 진성 팬 / 고인물 / 전설의 덕후 |
| 카테고리 | 카드 그리드 (2열) | 산리오 / 포켓몬 / 진격 / 귀멸 / 강철 + COMING SOON |
| 카드 | 카테고리 이미지 | assets/main-cards/{category}.png |
| 카드 | NORMAL / HARD 뱃지 | 완료 시 ✓ 녹색으로 변경 |
| 카드 | 카테고리명 | 이미지 하단 오버레이 텍스트 |
| 모달 | 난이도 선택 팝업 | 카드 클릭 시 등장 |
| 모달 | 노말 버튼 | 문제당 1점, 완료 시 비활성화 |
| 모달 | 하드 버튼 | 문제당 2점, 완료 시 비활성화 |
| 배경 | 픽셀 이모지 플로팅 | 별 / 꽃 / 하트 등 배경에서 떠다님 |

### 동작

- 카드 클릭 → 난이도 선택 모달 오픈
- 이미 완료한 난이도 버튼 → 비활성화
- 노말 / 하드 선택 → `quiz.html?category=xxx&mode=xxx` 이동
- 페이지 로드 시 localStorage 읽어 게이지 & 뱃지 자동 업데이트

## S02 — 퀴즈 진행 (quiz.html)

**역할:** 문제 출제, 타이머 운영, 정답 수집

### 구성 요소

| 영역 | 요소 | 설명 |
|------|------|------|
| 상단 | 진행 표시 | "3 / 10" 형태로 현재 문제 번호 |
| 상단 | 타이머 바 | 30초 → 0초, 10초 이하 시 빨간색 + 진동 애니메이션 |
| 상단 | 카테고리 & 난이도 | ex) "포켓몬 · HARD" |
| 문제 | 문제 카드 | YES/NO로 답할 수 있는 문장 |
| 버튼 | YES 버튼 | 정답이 true일 때 맞음 |
| 버튼 | NO 버튼 | 정답이 false일 때 맞음 |
| 결과 | 정답/오답 피드백 | 선택 직후 0.5초간 O / X 표시 후 다음 문제 |

### 동작

- URL에서 category, mode 읽기
- `data/{category}.js` 로드 → 해당 모드 20문제 중 랜덤 10개 추출
- 매 문제 30초 타이머 시작
- 시간 초과 → 자동 오답 처리 후 다음 문제
- YES / NO 선택 → 정답 체크 → 점수 누적
- 10문제 완료 → `result.html?category=xxx&mode=xxx&score=xxx` 이동

### 점수 계산

| 모드 | 정답 | 오답/시간초과 |
|------|------|--------------|
| 노말 | +1점 | 0점 |
| 하드 | +2점 | 0점 |

## S03 — 결과 페이지 (result.html)

**역할:** 점수 및 등급 표시, 답지 공개, 종합 덕력 현황

### 구성 요소

| 영역 | 요소 | 설명 |
|------|------|------|
| 결과 카드 | 카테고리 이미지 | assets/main-cards/{category}.png |
| 결과 카드 | 점수 | ex) "16 / 20" |
| 결과 카드 | 별점 | ★★★★☆ |
| 결과 카드 | 등급 타이틀 | IP별 커스텀 텍스트 (아래 표 참고) |
| 답지 | 10문제 전체 목록 | 맞힌 문제 ✅ / 틀린 문제 ❌ + 정답 표시 |
| 종합 | 덕력 게이지 | 현재까지 완료한 카테고리 누적 점수 |
| 종합 | 카테고리별 점수 | 완료한 것 점수 표시, 미완료는 🔒 |
| 버튼 | 공유 카드 만들기 | share.html로 이동 |
| 버튼 | 메인으로 | index.html로 이동 |

### IP별 등급 타이틀

| IP | 90%↑ | 70%↑ | 50%↑ | 30%↑ | 0%↑ |
|----|------|------|------|------|-----|
| 산리오 | 산리오 퓨로랜드 관장 | 헬로키티 친구 | 산리오 팬 | 굿즈만 아는 사람 | 쿠로미도 실망 |
| 포켓몬 | 포켓몬 마스터 | 체육관 관장 급 | 포켓몬 트레이너 | 포켓몬 고만 함 | 오박사도 너보단 낫다 |
| 진격 | 자유의 날개 | 조사병단 에이스 | 훈련병단 수료 | 벽 안에서 살아라 | 거인한테 잡아먹힘 |
| 귀멸 | 히노카미 신락 | 기둥 후보 | 갑대원 합격 | 최종선발 통과 | 최종선발 탈락 |
| 강철 | 현자의 돌 획득 | 국가 연금술사 | 연금술 수련생 | 등가교환 이해중 | 등가교환도 모름 |

### 동작

- URL에서 category, mode, score 읽기
- `localStorage.setItem('dk_{category}_{mode}', score)` 저장
- 정답률 계산 → 등급 결정
- 이전에 받은 문제/정답 데이터로 답지 렌더링

## S04 — 공유 카드 (share.html)

**역할:** 결과를 SNS 공유용 이미지 카드로 변환

### 구성 요소

| 영역 | 요소 | 설명 |
|------|------|------|
| 공유 카드 | 서비스 로고 | "덕력 감별소" |
| 공유 카드 | 카테고리 + 난이도 | ex) "포켓몬 · HARD" |
| 공유 카드 | 캐릭터 이미지 | assets/main-cards/{category}.png |
| 공유 카드 | 점수 | ex) "16 / 20" |
| 공유 카드 | 등급 타이틀 | ex) "체육관 관장 급" |
| 공유 카드 | 별점 | ★★★★☆ |
| 공유 카드 | 한 줄 멘트 | 등급별 고정 멘트 |
| 버튼 | 이미지로 저장 | html2canvas로 카드 캡처 후 PNG 다운로드 |
| 버튼 | 링크 복사 | 현재 URL 클립보드 복사 |
| 버튼 | 메인으로 | index.html 이동 |

### 동작

- URL에서 category, mode, score, grade 읽기
- 카드 UI 렌더링
- 이미지 저장: `html2canvas` 라이브러리로 `.share-card` 캡처 → PNG 다운로드
- 링크 복사: `navigator.clipboard.writeText(location.href)`
- 카드 권장 비율: **9:16** (인스타 스토리 사이즈)
