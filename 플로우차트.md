# 덕력 감별소 — Flow Chart

---

## 전체 사용자 흐름

```mermaid
flowchart TD
    A([🌸 서비스 접속]) --> B[메인 페이지\nindex.html]

    B --> C[덕력 게이지 확인\n누적 점수 표시]
    B --> D[카테고리 카드 선택\n산리오 / 포켓몬 / 진격 / 귀멸 / 강철]

    D --> E{이미 완료한\n난이도 확인}

    E -->|노말 미완료| F[노말 선택\n문제당 1점]
    E -->|하드 미완료| G[하드 선택\n문제당 2점]
    E -->|둘 다 완료| H[버튼 비활성화\n재응시 불가]

    F --> I[퀴즈 진행\nquiz.html]
    G --> I

    I --> J[문제 출력\n총 10문제 랜덤]
    J --> K{30초 타이머}

    K -->|시간 초과| L[자동 오답 처리]
    K -->|YES 선택| M{정답 확인}
    K -->|NO 선택| M

    M -->|정답| N[점수 누적\n노말+1 / 하드+2]
    M -->|오답| O[0점]

    L --> P{10문제\n완료?}
    N --> P
    O --> P

    P -->|아직 남음| J
    P -->|완료| Q[결과 페이지\nresult.html]

    Q --> R[점수 / 등급 표시\nIP별 커스텀 타이틀]
    Q --> S[10문제 답지 공개\n맞힘 ✅ / 틀림 ❌]
    Q --> T[localStorage 저장\ndk_category_mode = 점수]
    Q --> U[종합 덕력 현황\n완료한 카테고리 누적]

    T --> V{공유할까?}
    V -->|공유하기| W[공유 카드\nshare.html]
    V -->|메인으로| B

    W --> X[결과 카드 렌더링\n카테고리 / 점수 / 등급]
    X --> Y[이미지로 저장\nPNG 다운로드]
    X --> Z[링크 복사\n클립보드]
    Y --> B
    Z --> B
```

## 데이터 흐름

```mermaid
flowchart LR
    subgraph 데이터 소스
        D1[data/sanrio.js]
        D2[data/pokemon.js]
        D3[data/aot.js]
        D4[data/kimetsu.js]
        D5[data/fma.js]
    end

    subgraph URL 파라미터
        P1["?category=pokemon\n&mode=hard"]
        P2["?category=pokemon\n&mode=hard\n&score=16"]
        P3["?category=pokemon\n&mode=hard\n&score=16\n&grade=고인물"]
    end

    subgraph localStorage
        L1[dk_pokemon_normal]
        L2[dk_pokemon_hard]
        L3[dk_sanrio_normal]
        L4[... 총 10개 키]
    end

    D1 & D2 & D3 & D4 & D5 --> quiz.html
    quiz.html -->|P1| quiz.html
    quiz.html -->|P2| result.html
    result.html -->|저장| localStorage
    result.html -->|P3| share.html
    localStorage -->|읽기| index.html
```

## 화면 전환 요약

```
[ index.html ]
      │
      │ 카테고리 선택 + 난이도 선택
      ▼
[ quiz.html?category=xxx&mode=xxx ]
      │
      │ 10문제 완료
      ▼
[ result.html?category=xxx&mode=xxx&score=xxx ]
      │                    │
      │ 공유하기            │ 메인으로
      ▼                    ▼
[ share.html ]      [ index.html ]
      │
      │ 메인으로
      ▼
[ index.html ]
```
